Introduction

The second tuning method developed by Ziegler and Nichols did not even rely on the presence of a controller, in contrast to the first tuning method they introduced in their seminal 1942 paper, in which the process was made to oscillate using proportional-only automatic control, and the variables of that oscillation served to define PID tuning variables. With the second method, the control element (valve) was manually "stepped-changed," and the effect it had on the process variable was then investigated. After the step-change in the output signal is produced with the controller in manual mode, the dead time and the reaction rate are two key components that are thoroughly examined in the process variable trend. Dead time (L) is the duration between an output step-change and the initial signal of a process variable change (Note 1). The reaction rate is the fastest rate at which a process variable can change following a step-change in output (the maximum time-derivative of the process variable)

Figure 1 Zn method open loop

The reactions that integrating and self-regulating processes display are dead time and reaction rate. The process variable will climb at a maximum rate following the output step-change, and this rate of rise will constitute the process' reaction rate, regardless of whether it ultimately stabilizes at a new value. A problem with the Ziegler-Nichols open-loop method is that it completely overlooks the distinction between integrating and self-regulating process types. This is a problem, as our analysis of each process type has indicated. Dead time in a process tends to remain constant regardless of how much the output changes, but reaction rate normally fluctuates in direct proportion to the amount of the output step-change. For instance, a 10% output step-change will often cause the PV to rise at a rate that is twice as high as the impacts of a 5% output step-change. To make sure that our prediction computations only take into account the process' inherent characteristics and not our own arbitrary open-loop tuning operations, we also need to take into account the output step-change magnitude (m).

Conclusion

The PID controller is the backbone of modern process control systems. The proportional, integral, and derivative control modes each have a distinct function. Proportional and integral control modes are needed for the majority of control loops, whereas derivative is only occasionally used. Using various Kp, Ki, and Kd values, we test the system's responsiveness in this lab experiment. The graph demonstrates that the derivative mode produces more control action when the error changes more quickly. If the error remains the same, the derivative action is zero. The larger the derivative time setting, the more derivative action is performed. When the derivative time is set to 0, this option is essentially disabled. We can also see that employing root locus enhances system design with more precision. If the derivative time is set too long, the control loop will run in an unstable manner. It assists in examining the system's long-term stability. Using root locus plots, we can calculate the system's gain margin, relative stability, phase margin, and settling time.

If the root locus branches move to the right, the system is significantly less stable; if they move to the left, the system is somewhat more stable. The Ziegler-Nichols tuning rule gives the best disturbance rejection performance for your PID loops. With this level, command tracking performance is frequently not very good. It is simpleand you might not even be aware of itto select a tuning rule that is insufficient for the application.
